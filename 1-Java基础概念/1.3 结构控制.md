# 1.条件控制

1. if
2. if...else
3. if...else if...else if...else

>要求越详细的分支往前放，要求越宽泛的分支往后放

```java
if (num % 5 == 0 && num % 6 == 0)  
    System.out.println("可以被5和6同时整除");  
else if (num % 5 == 0)  
    System.out.println("可以被5整除");  
else if (num % 6 == 0)  
    System.out.println("可以被6整除");  
else  
    System.out.println("不能被5和6整除");
```

### 循环的中断
- *break*：中断所有的循环
- *continue*：中断当前的循环，之后的循环继续
>只能影响单层循环
